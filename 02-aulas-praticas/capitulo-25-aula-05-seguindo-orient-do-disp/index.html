ANOTAÇÕES DA AULA

Nessa aula, nós aprendemos a como definir as media queries para a orientação das nossas telas de em um único arquivo de CSS. No caso dessa aula, nós fizemos isso dentro do próprio HTML.
Mas você pode colocar isso em um arquivo CSS aparte, basta inserir a tag: 

 
        @media screen and (orientation: landscape(horizontal) ou portrait(vertical)){ 
            
            body{
                background-color: red;
                atributos....
            }

        }


Fato interessante: dentro das media queries, você precisa colocar as tags dos elementos que você irá mexer, ok? Por exemplo, se vc colocou um @media screen and (orientation: portrait) e quer definir como o corpo e os títulos se comportarçao, use as tags desses elementos: h1 e body!
