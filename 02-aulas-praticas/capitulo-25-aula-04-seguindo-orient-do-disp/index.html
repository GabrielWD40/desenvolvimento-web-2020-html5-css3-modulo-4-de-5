ANOTAÇÕES DA AULA

Como você pode ver, nessa aula nós usamos duas media queries só que com as features diferentes. As features nesse caso, serão as orientações da tela. Usamos duas, uma para a rotação na horizontal e outra para a vertical.

Se liga, na hora que mudarmos a orientação da tela, a imagem também muda de maneira automática. Usamos as imagens deo professor Gustavo para realizar esse feito!

Outra coisa que nós fizemos foi colocar o background-size: contain. Para que ele coloque a imagem toda dentro do background. Se colocarmos a propriedade "cover", acaba ficando bem zoado. Também deixamos os atributos de posição bem coladinhos nas bordas para que não hajam quebras e trechos zoados!

Além disso, nós também fizemos a seguinte mudança: Se algum trecho do código irá se repetir, é possível colocá-lo dentro do estilo principal, que nesse caso é o nosso media="all"!

Gira a tela aí, Lecoti!! Veja que o background vai mudar de acordo com a posição da media query por conta das medias querys!
